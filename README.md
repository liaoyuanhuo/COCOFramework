COCOFramework
=============

My android development framework, under heavy development.
Short code, rapid development, less bug, more fun


How to start
=============
```xml
    compile('com.cocosw:framework:+@aar') {
        transitive = true
    }
    debugCompile('com.cocosw.framework:debug:+@aar') {
        transitive = true
    }
```
